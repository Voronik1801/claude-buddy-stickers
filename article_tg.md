# Статья для @aishipuchka (TG)

---

У Claude Code есть секретная фича — **buddy**. Маленький ASCII-питомец, который живёт в твоём терминале и комментирует твой код.

У меня Wobble — терракотовый чонк с буддийским принятием. Терпение 94, отладка 17. Он просто *вибрирует* когда видит мои баги.

Я сделала штуку: скилл, который **автоматически генерирует стикерпак в Telegram** из твоего buddy.

Вот что происходит:
1. Скрипт читает `~/.claude.json` — кто твой buddy, какой вид, какие статы
2. Подбирает 12 уникальных эмоций **под твой вид** (у гуся — honk/steal/chase, у кота — loaf/zoomies/knock, у моего чонка — zen/vibrate/meditate)
3. Генерирует картинки через AI (Gemini через OpenRouter)
4. Убирает фон, ресайзит до 512px
5. Создаёт стикерпак в Telegram через Bot API

**Одна команда — и у тебя стикерпак с твоим buddy.**

18 видов поддерживается. Duck, Goose, Cat, Dragon, Axolotl, Ghost, Robot, Capybara...

А ещё — buddy может **реагировать стикерами** на твою работу:
- Тесты прошли? Buddy шлёт 👍
- Билд упал? Buddy шлёт 🤦
- Деплоишь в прод? Buddy шлёт 🔥
- Долгая установка? Buddy медитирует ✨

Всё в открытом доступе: github.com/Voronik1801/claude-buddy-stickers

Если у тебя есть Claude Code — попробуй. `/buddy stickers` и го.

Покажи своего buddy в комментах 👇
